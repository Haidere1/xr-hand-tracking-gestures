# XR Hand Tracking & Gestures Plugin

A **drop-in Unreal Engine plugin** for **hand tracking and gesture recognition** using **OpenXR**.  
Tested on **Varjo XR-4**, but works with all headsets that support **OpenXR runtime and OpenXR hand tracking**. Easily integrate into your project — fully functional
and ready to use.

---

## Features

- Real-time hand tracking using OpenXR  
- Gesture recognition  
- Drop-in plugin — ready to use  
- Lightweight and optimized for VR/AR projects  
- Tested on **Varjo XR-4**; compatible with all OpenXR hand-tracking devices  

---

## Requirements

- Headset with OpenXR runtime  
- Hand tracking support enabled in your runtime  

---

## Installation

1. **Download or clone** this repository.  
2. Copy the plugin folder into your Unreal project's `Plugins` directory.  
3. Open your project in Unreal Engine.  
4. Enable the plugin in **Edit → Plugins → Installed → XR Hand Tracking & Gestures**.  
5. Restart the editor if prompted.  

---

## Usage

1. Drag and drop the provided components into your scene.  
2. Configure hand tracking and gesture settings in the plugin menu.  
3. Build and run your project on a compatible headset.  

The plugin is designed to work **out-of-the-box**, so you can start tracking hands and recognizing gestures immediately.  

---

## Supported Headsets

- Tested on **Varjo XR-4**  
- Compatible with any headset that supports **OpenXR runtime and hand tracking**
